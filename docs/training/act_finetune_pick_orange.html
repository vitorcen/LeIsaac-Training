ACT 微调记录

SO-101 PickOrange · LeIsaac benchmark baseline #1 · 2026-05-18

Action Chunking Transformer (Tonyzhao et al. 2023, ALOHA paper) — the simplest and cheapest baseline in the leaderboard: ~80M params, no VLM, no diffusion, just a ResNet18 + Transformer encoder-decoder predicting an action chunk in one forward pass.

0. TL;DR

ACT 是 leaderboard 里最便宜的 baseline（80M params，~10ms/chunk forward）。 本地训出来的 ckpt wsagi/ACT-PickOrange 在 strict eval 下 0/3 round, 2/9 oranges, avg 130s（横评中位）， 关键发现是 chunk-execution horizon trap：默认 horizon=16 跑不动，必须 h=32 才能复现 demo 抓取行为。

1. 架构核心

组件来源实现参数估算
Vision backbonetorchvision ResNet18vision_backbone="resnet18" + ImageNet pretrained；可选 final stride→dilation；feature map flatten + projection 到 dim_model~11M
Transformer encoderACT 自研（modeling_act.py:514）4 层标准 multi-head self-attention，非 causal（全 bidirectional）；输入：[CLS, image patches × N_cam, state, (训练时) latent z]~10M
Transformer decoderACT 自研（modeling_act.py:572）7 层 decoder block；query = learnable positional embedding（chunk_size 个）；KV = encoder output；输出每个 query → 1 个 action~25M
CVAE encoder（训练 only）ACT 自研训练时把 GT action chunk 编码成 latent z；推理时 z = 0；提供"action prior"正则~10M
合计——~80M（全部 trainable，无 LoRA）

读图要点：

  • 没有 VLM：ACT 是纯 vision-conditioned policy，不接 language。任务身份由训练 dataset 决定，inference 时无 prompt
  • CVAE 是 trick 不是核心：训练时把 GT action 编成 latent z 提供正则，推理时 z=0；理论上简化版（去掉 CVAE）也能跑，paper 报 z=0 + KL 退火帮助 small-data 收敛
  • 单 forward 出整个 chunk（chunk_size=100）：跟 ACT/DP/SmolVLA 一脉相承，但 ACT 是 直接 regression，无 diffusion，最便宜。10ms 跑完一个 100-step chunk = 等效 0.1ms/action
  • Temporal ensembling（可选）：每 sim step 都 forward，把多个过去 prediction 加权求平均 chunk[0] → 抗噪；我们 benchmark 没开，开后 latency 会上去

3. Eval 参数 + chunk-execution horizon trap

🔴 horizon trap：默认 --policy_action_horizon=16（消费 chunk 前 16 步就重 query）跑不动，arm 漂浮无意义。必须 horizon=32 才能复现 demo 行为。
eval 参数值说明
step_hz30对齐 dataset 30Hz
policy_action_horizon32（不是默认 16）chunk=100 但只消费前 32 → 高频重规划
episode_length_s120
eval_rounds3strict mode
peak VRAM~8-10 GB最小的 baseline
avg latency~10 ms / chunk forward等效 0.1ms / action
